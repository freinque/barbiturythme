Detailed Description

reads, preprocesses and processes an audio signal to extract its main rhythmic features.

Onset detection using 2 algorithms (spectral flow and phase spectra) written in the modal python module. Actually better at detecting percussive instruments, although fancier onset detection techniques could be used.
